# Portfolio

This project uses Vue 3 in Vite, vueuse, gsap.



## Project Setup

### Requirements
- Docker
- npm

### Compile and Hot-Reload for Development



```sh
cd vue 
```

```sh
npm install
```

```sh
cd .. 
```

```sh
docker compose up -d --build
```
By default, app will be available at: http://localhost:8080/
